# Recruiter Summary

The RedRack Technologies SOC Lab is a hands-on enterprise cybersecurity environment designed to demonstrate infrastructure administration, security monitoring, SIEM engineering, automation, threat hunting, and incident response capabilities.

The lab includes:
- Windows Server 2022 Active Directory infrastructure
- Windows 10 monitored endpoints
- Ubuntu 24 automation systems
- Sysmon endpoint telemetry
- PowerShell logging
- SIEM integrations using Wazuh and Splunk
- Python and Bash automation
- Threat hunting workflows
- MITRE ATT&CK mapping
- Monitoring and operational documentation

This repository documents the build process, validation procedures, troubleshooting, automation workflows, and operational practices used throughout the environment.
